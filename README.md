# HASSOM

Projeto da cadeira de **Projeto de Sistemas de Informação**

### Descrição

> Loja de instrumentos musicais.
> Back end

### Tecnologias 

* NestJS
* TypeORM
* Postgresql
* JWT (Json Web Token)
* Express
* TypeScript

## Instalação

``` bash
# development
$ npm run start
````  
````bash
# watch mode
$ npm run start:dev
````  
````bash
# production mode
$ npm run start:prod
````
